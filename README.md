# 🤖 Telegram Software Library Bot

بوت تيليجرام متقدم يعمل كمكتبة برامج رقمية، يتيح للمستخدمين البحث عن البرامج والملفات وتحميلها بسهولة.

## ✨ المميزات

### 🔍 البحث الذكي
- بحث بالاسم الكامل أو الجزئي
- بحث بالكلمات المفتاحية
- اقتراحات تلقائية
- تصحيح أخطاء الكتابة
- ترتيب النتائج حسب الشعبية
- بحث غير حساس لحالة الأحرف

### 📦 إدارة الملفات
- فهرسة تلقائية للملفات
- تصنيف حسب النوع والفئة
- دعم جميع أنواع الملفات
- تخزين البيانات الوصفية
- نظام كلمات مفتاحية ذكي

### 👥 تجربة المستخدم
- واجهة تفاعلية سهلة
- أزرارInline للتنقل
- نظام تقييم بالنجوم
- قائمة مفضلة
- سجل التحميلات
- مشاركة البرامج

### 📊 لوحة تحكم المشرف
- إحصائيات شاملة
- إدارة المستخدمين
- حظر/فك حظر
- إرسال رسائل جماعية
- إدارة النسخ الاحتياطي
- وضع الصيانة

### 🛡️ الأمان
- Rate Limiting
- Anti-Spam
- Flood Protection
- SQL Injection Protection
- Input Validation

## 📋 المتطلبات

- Python 3.12+
- pip (مدير حزم Python)
- Git (اختياري)
